Cryptic Carnage

Cryptic carnage is a 3D FPS zombie hoarde shooter game made within unity the base of object of the game is to see how long you can last and how high you can get your score. The game uses raycasting to detect damaging and hits, using animation event systems to deal player damage, Nav Meshes for player controll and AI zombie behavior, lerptimers for regenerating health and Unitys UI system to display timer, ammo, health, and score.
